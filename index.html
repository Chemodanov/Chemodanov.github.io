Обзор программы шифрования PGP Desktop

Введение

PGP — компьютерная программа позволяющая выполнять операции шифрования и цифровой подписи сообщений, файлов и другой информации, представленной в электронном виде.

Главное окно

В главном окне находится несколько вкладок, отвечающих за различные режимы работы программы:

  1. PGP Keys
  2. PGP Messaging
  3. PGP Zip
  4. PGP Disk
  5. PGP Viewer
  6. PGP NetShare

PGP Keys

Активна по умолчанию. В данной вкладке пользователь видит ранее созданные ключи, которые впоследствии будут использоваться для шифрования различной информации. Чтобы добавить новый ключ, необходимо выполнить операцию File > New PGP key.

PGP Messaging

Здесь пользователь работает с электронной почтой. Возможно автоматически шифровать сообщения с указанного адреса электронной почты, по заданному ключу. В нижней половине вкладки пользователь может настраивать различные политики безопасности.

PGP Zip

Вкладка PGP Zip позволяет шифровать файлы. Делается это просто: необходимо перетащить нужные файлы в поле окна New PGP Zip, после чего указать тип и ключ шифрования. По умолчанию мы видим во вкладке историю успешных верификаций.

PGP Viewer

Осуществляет расшифровку входящей электронной почты. Для этого достаточно перенести email-сообщение в указанную область.

PGP NetShare

С помощью функционала этой вкладки осуществляется разрешение на доступ к содержимому указанных папок по локальной сети.

Вывод

Программа PGP дает широкий набор возможностей для шифрования данных. При этом она имеет понятный интерфейс с несколькими особенностями, облегчающими работу с ней, например частое использование drag&drop для добавления файлов. Недостатком является интерфейс на английском языке, требующий базовые знания в нем.

Обзор сайта lolking.net

В данном обзоре будет описан один из крупнейших сайтов по одной из самых популярных игр в мире league of legends - lolking.net.

Главная форма

Главная форма сайта содержит:

  1. Меню для навигации по сайту
  2. Многофункциональную поисковую строку
  3. Список наиболее популярных онлайн-трансляций

  4. Новостую ленту

    Основные пункты главного меню

    Lolking now

Позволяет получить полную информацию о любом матче, введя имя любого из его участников.

Charts(Рейтинги)

В этом разделе отображается различная статистическая информация по игре.

Streams(Трансляции)

В этом разделе отображается список трансляций игроков, желающих выставить свою игру на всеобщее обозрение.

Champion Guides(Руководства по героям)

Данный раздел позволяет просматривать руководства по героям от других игроков, а также создавать свои.

Поисковая строка

С помощью поисковой строки пользователи сайта могут найти:

  1. Информацию об учетной записи любого игрока
  2. Информацию об учетных записях соперников любого игрока, если в данный момент он играет.
  3. Различную информацию о любом герое.

Онлайн-трансляции

5 трансляций с наибольшим колличеством зрителей на данный момент.

Вывод:

Lolking.net даёт доступ к огромному колличеству информации по игре League of Legends. При этом он обладает простым для понимания интерфейсом, из-за наглядного представления информации(за исключением новостей, текстовая информация дублируется в виде изображений, а статистическая информация в виде графиков и таблиц). Недостатком для русскоговорящего пользователя может стать то, что сайт можно просматривать только на английском языке, однако для понимания большинства контента требуется лишь базовый уровень английского, и только раздел новостей может вызвать какие-то затруднения в переводе.
